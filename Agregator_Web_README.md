# Агрегатор управления электропотреблением

## Описание проекта

Веб-приложение для агрегации управления электропотреблением, позволяющее потребителям электроэнергии взаимодействовать с агрегаторами. Система предоставляет функционал для регистрации пользователей (физических лиц) и агрегаторов, управления подписками, загрузки данных о потреблении электроэнергии и визуализации статистики.

## Архитектура

Проект построен на **Flask** с использованием **SQLAlchemy** в качестве ORM. Основные компоненты:

- **Аутентификация и авторизация** — раздельная регистрация/вход для пользователей и агрегаторов
- **Основной модуль** — управление подписками, профилями, отображение данных
- **CLI команды** — загрузка регионов и данных о потреблении
